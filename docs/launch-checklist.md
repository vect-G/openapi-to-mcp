# Launch Checklist

## Before publish

- Add project logo and social preview image.
- Record a 20-40s demo GIF:
  - openapi-to-mcp list from remote URL
  - copy output into MCP client config
  - show one successful tool call
- Add badges in README (PyPI, tests, stars).
- Create a concise one-line value proposition in repo description.

## Launch day

- Post to X, Reddit (`r/MachineLearning`, `r/LocalLLaMA`, `r/Python`), and Hacker News.
- Lead with a concrete before/after workflow.
- Share a minimal "try now" command.

## Week 1 follow-up

- Reply quickly to issues and PRs.
- Ship at least one visible update within 48 hours.
- Add real-world examples from early users.

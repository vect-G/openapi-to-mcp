# Launch Post Template

Built `openapi-to-mcp`: one command to turn OpenAPI/Swagger into a runnable MCP server.

Why this matters:
- Most teams already have OpenAPI specs.
- AI agents need MCP tools.
- This bridges both in minutes.

Try it:
```bash
openapi-to-mcp run https://petstore3.swagger.io/api/v3/openapi.json
```
